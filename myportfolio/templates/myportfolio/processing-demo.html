Description

This visualization demonstrates an insertion sort algorithm, processed in live-time by your browser in JavaScript. The program works as follows:

  1. Generate random values and store them into an array
  2. Iterate through the array and create a line with height equal to the array element
  3. Iterate through the array and apply the insertion sort algorithm
